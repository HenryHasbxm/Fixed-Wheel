# Rigged Wheel

The background color is customizable.

Names are put one on each line (return after each name)

Put a comma at the end of the name you want to win to rig the wheel and spin on the name requested.
There is a lever to toggle on and off the rig. Off will revert the wheel to random and On will turn on the rig.
Please remember to turn on rigging and put the comma for the rig to work.
